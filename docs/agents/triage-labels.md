# 分类标签

各 skill 使用五个标准分类角色。本文件将这些角色映射到本仓库 issue 追踪器中实际使用的标签名称。

| 技能中的标签             | 实际使用的标签       | 含义                           |
| ------------------------ | -------------------- | ------------------------------ |
| `needs-triage`           | `needs-triage`       | 需要维护者评估                 |
| `needs-info`             | `needs-info`         | 等待提交者补充更多信息         |
| `ready-for-agent`        | `ready-for-agent`    | 规格明确，AFK agent 可接手     |
| `ready-for-human`        | `ready-for-human`    | 需要人工实现                   |
| `wontfix`                | `wontfix`            | 不会处理                       |

当 skill 提到某个角色（例如"应用 AFK-ready 分类标签"）时，使用本表中对应的标签名称。

如果你需要更改实际的标签词汇，修改右侧列即可。
