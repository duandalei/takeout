# Triage Labels（分类标签）

技能使用五个标准 triage 角色来描述 issue 状态。本文件将这些角色映射到本仓库实际使用的标签名称。

| 技能中的标签名称   | 本仓库标签名称 | 含义                           |
| ----------------- | ------------- | ------------------------------ |
| `needs-triage`    | `needs-triage`    | 维护者需要评估此 issue     |
| `needs-info`      | `needs-info`      | 等待报告者提供更多信息       |
| `ready-for-agent` | `ready-for-agent` | 已充分描述，可供 agent 拾取    |
| `ready-for-human` | `ready-for-human` | 需要人工实现                  |
| `wontfix`         | `wontfix`         | 不会处理                      |

当技能提到某个角色时（例如"应用 AFK-ready triage 标签"），请使用本表中对应的标签名称。

可根据实际使用的标签词汇修改右侧列。
